#Ejecutar en Desarrollo

1. Clonar en Repositorio
2. Ejecutar
```
yarn install
```

3. Tener Nest CLI instalado
```
npm i -g @nest/cli
```

4. Levantar la Base de Datos
```
docker-compose up  -d
```
##Stack Usado
* MongoDB
* Nest
